Detailed Description

For selecting a decoder given a video file. Instead of providing hard-coded mappings from mime-type and profile to a decoder, this class searches the component store and returns a list of valid codecs that are able to play back the given video.

◆ getTracksByType()

List< Track > getTracksByType ( CodecType  type)
private

Returns a list of tracks which are of a certain mime prefix. Specifically if codec type is VIDEO then all tracks with a mime type starting with video will be returned. The method is not case sensitive, both video and Video will match. The given mime type is converted to lowercase before comparison. A list of media format objects that represent tracks

Parameters
typeA flag for the type of track.
Returns
A list of tracks that are of the requested type.
