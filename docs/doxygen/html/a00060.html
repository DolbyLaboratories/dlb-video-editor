Detailed Description

Configures image pipeline for Dolby Vision editing. This class creates ImageReader and ImageWriter objects that support Dolby Vision, which app developers can retrieve and access in their editing code. This class configures the appropriate output DataSpace and hardware buffer format for both transcoding and video previewing.

This constructor builds the ImageReader and ImageWriter objects. The reader and writer are ready for use after the constructor returns.

Parameters
inputSizeThe dimensions of the input video.
outputSizeThe dimensions of the output surface.
outputSurfaceThe destination Surface object.
previewModeTrue if the output surface is writing to a preview, False if transcoding.
inputProfileThe profile of the input video, as specified by MediaFormat.
encoderFormatThe intended format of the output ("HEVC", "AVC, or "DV-ME").
